How to setup Raspberry Pi

1. Use Balena etcher to install and setup Raspberry Pi with strech OS 

2. Enable SSH from Raspberry Pi ssh menu
 -  Preferences - Interfaces - Enable ssh 

3. Setup docker with `curl -fsSL get.docker.com -o get-docker.sh && sh get-docker.sh` 
    - Something about curling and installing a random script
    https://medium.freecodecamp.org/the-easy-way-to-set-up-docker-on-a-raspberry-pi-7d24ced073ef

4. Install k3s for docker
